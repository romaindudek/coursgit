Git, c'est quoi et ça sert à quoi ?

Git est un logiciel de gestion de versions décentralisé.

  • Un logiciel ? Donc, un logiciel nécessite une installation...
  • Un logiciel avec quelle interface ?
  • Pourquoi j'ai besoin de gérer des "versions" ?
  • Je travaille en équipe ou tout seul, je l'utilise pareil ?

Du coup, GitHub, c'est quoi?

GitHub (/ɡɪt hʌb/ ; exploité sous le nom de GitHub, Inc.) est un service web d'hébergement et de gestion de développement de logiciels, utilisant le logiciel de gestion de versions Git.

  • Un Service ? Donc, c'est payant...
  • Est-ce qu'on peut utiliser Git sans GitHub ?
  • Il n'existe que github ou il y a des alternatives ?

Sous windows

Installation de Git BASH... sur le site https://gitforwindows.org/

Sous mac

Dans un terminal, taper l'instruction :

$ git --version

Si l'instruction ne donne pas de résultat :

$ brew install git

Attention : Si brew n'est pas installé, il faut penser à installer brew : site Brew

Sous les systèmes linux (ubuntu)

$ git --version

Si l'instruction ne donne pas de résultat :

$ sudo apt-get install git
